# Projeto de deploy PosTech - Hackathon

Projeto para deploy da aplicação desenvolvida no Hackathon PosTech FIAP

## Deploy Terraform

Terraform para deploy da infraestrutura AWS da aplicação

### Executar 

- terraform init
- terraform plan
- terraform apply
